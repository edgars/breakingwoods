Breakinwoods - A set of additional components for JBoss ESB

This is a project that aims to be a repository of new or additional components, as well as a forge for new ideas for JBoss ESB, which would make the things even more easier when you need an integrating solution. The components here are straightforward to use, and you can contribute even with ideas, comments or with your component as well, just start today to request about a new component that we might be able to create a new component, extensions or tools. The components made here may be suitable either by JBoss ESB Community version or in JBoss SOA Platform

Dependencies

JBoss ESB, JBoss jBPM, Drools and others
